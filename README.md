# ✋ Gesture Controlled Scientific Calculator

A Gesture-Controlled Scientific Calculator that allows users to perform mathematical calculations using hand gestures detected through a webcam. The system uses Computer Vision techniques with OpenCV to recognize gestures and convert them into calculator inputs in real time.

---

## 🚀 Project Overview

Traditional calculators require physical interaction using buttons. This project introduces a touchless calculator interface where users can perform calculations using hand gestures captured by a webcam.

The system processes real-time video input, detects gestures, and maps them to calculator functions such as numbers, arithmetic operations, and scientific calculations.

---

## 🧠 Technologies Used

- Python
- OpenCV

- Computer Vision

## ⚙️ Features

- Real-time webcam input
- Hand gesture detection
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, etc.)
- Interactive virtual calculator interface
- Real-time result display

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/gesture-controlled-calculator.git
```

2. Navigate to the project directory

```bash
cd gesture-controlled-calculator
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run the Project

Run the main Python file:

```bash
python main.py
```

After running the program, the webcam will open and the gesture-controlled calculator interface will appear.
